# 📊 Dashboard de Faturamento - Power BI

## 📌 Sobre o Projeto
Dashboard desenvolvido no Power BI com foco em análise de faturamento e indicadores comerciais. O projeto apresenta métricas estratégicas para auxiliar na visualização e tomada de decisão baseada em dados.

## 🚀 Indicadores Apresentados
- Faturamento Geral
- Quantidade de Pedidos
- Quantidade de Clientes
- Faturamento por Estado
- Faturamento por Ano
- Modelos Mais Vendidos
- Modelos Vendidos por Cor
- Fretes por Fabricante

## 🛠️ Ferramentas Utilizadas
- Power BI
- DAX
- Modelagem de Dados
- Visualização de Dados

## 🎯 Objetivo
Transformar dados em informações visuais e interativas para análise de desempenho comercial e apoio à tomada de decisão.
